# Digital Twin

## Overview
This repository focuses on generating realistic synthetic traffic datasets and scenario generation, annotation, and analysis in digital twin environments.

The project has **Digital twin simulation** for realistic environment and traffic generation  


## Repository Structure
- `README.md` — project description
- `data/` — generated datasets and outputs
- `sionna_csi_prediction.py` — CSI generation and prediction workflow
- `sionna_traffic_prediction.py` — traffic generation and prediction workflow

## Privacy
No personally identifiable information (PII) is used in this project. All data is either fully synthetic, or anonymized and aggregated

## Use Cases
This repository can be used for:
- synthetic traffic generation
- scenario prototyping in digital twin environments
- traffic and CSI prediction experiments
