# -traffic-twin-llm
Title: Traffic Data Digital Twin + LLM Research
Goal: Generate realistic synthetic traffic datasets and study LLM-assisted scenario generation/annotation/analysis.
Data: Trajectories, speeds, flows, signal timing, incidents (simulated).
Methods: Simulation digital twin + LLM prompting/tools for scenario design and labeling.
Privacy: No PII; only synthetic or anonymized aggregated data.
